Lấy Telegram ID

App sẽ tự lấy Telegram ID từ bot và điền lại vào ô bên ngoài.

Cách lấy Telegram ID

  1. 1. Bấm Mở bot Telegram hoặc copy link bên dưới để gửi cho người cần lấy Telegram ID.
  2. 2. Người nhận mở link đó rồi bấm Start bằng đúng tài khoản Telegram muốn nhận thông báo.
  3. 3. Quay lại popup này, app sẽ tự nhận Telegram ID và điền vào ô bên ngoài.
  4. 4. Muốn ngắt thông báo, chỉ cần xóa trắng ô Telegram ID rồi lưu lại.
  5. 5. Muốn thay sang tài khoản khác, bấm lại Lấy Telegram ID để tạo link mới.
